## Usage

```bash
$ git clone https://github.com/IgorSouzaBezerra/ProductManagement.git
```

### Install Dependency Yarn or Npm
```bash
$ yarn install
$ npm install
```

### Database Connection

in the file 
**src/shared/infra/http/mongo/index.ts**
insert connection string with mongo
